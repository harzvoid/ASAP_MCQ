ASAPMCQ is an AI-powered learning platform that transforms any study topic into personalized, exam-ready multiple-choice questions within seconds. Built using FastAPI and Google's Gemini AI, it helps students generate high-quality quizzes tailored to their learning needs. Users can customize difficulty levels, question types, and the number of questions for focused practice sessions. The platform is designed to support preparation for school examinations, university semester exams, competitive government exams, engineering entrance tests like GATE, and placement assessments. With an intuitive interface, real-time AI generation, and detailed performance analytics, ASAPMCQ makes revision faster, smarter, and more engaging. The goal is to provide an accessible and intelligent study companion that adapts to every learner's requirements, improving knowledge retention and exam confidence through personalized practice.


## ✨ Features

- 🤖 AI-powered MCQ generation using Google Gemini 2.5 Flash
- 📚 Generate personalized questions from any topic or subject
- 🎯 Adjustable difficulty levels (Easy, Medium, Hard)
- 🔢 Choose the number of questions per quiz
- 📝 Supports multiple question formats
  - Multiple Choice
  - Multi-Select
  - True / False
  - Fill in the Blank
- ⚡ Beautiful AI loading animation
- ⏱️ Interactive timed quiz interface
- 📊 Detailed performance analytics
- 📈 Topic-wise performance insights
- 🔄 Instant quiz review with explanations
- 🌙 Modern responsive UI
- 🚀 Fast AI-powered question generation
